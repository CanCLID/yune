ignored TypeDuck-Web checkout into the tracked `apps/yune-web/` Vite app.

Do not regenerate this patch for normal `yune-web` work. Edit the tracked app
root directly:

- `apps/yune-web/src/`
- `apps/yune-web/public/`
- `apps/yune-web/index.html`
- app-local config and package files
